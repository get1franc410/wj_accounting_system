PostgreSQL Installation Guide

  1. Download PostgreSQL from postgresql.org
  2. Run the installer as Administrator
  3. Follow the installation wizard
  4. Remember the password you set for the 'postgres' user
  5. Default port is 5432 (keep this unless you have conflicts)
  6. After installation, create a database for your accounting system

After PostgreSQL installation:

  1. Open pgAdmin (PostgreSQL management tool)
  2. Connect to your PostgreSQL server
  3. Right-click on "Databases" → "Create" → "Database"
  4. Name it "accounting_system" (or your preferred name)
  5. Click "Save"
Alternative: Use command line:
createdb -U postgres accounting_system
